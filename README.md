# Ytdownloader
A simple yt downloader build with ytdl-core and electron.
Use it as u want.

For Bugs and suggestions please open an Issue.

## Build
```
git clone https://github.com/Redstonerayy/ytdownloader
npm install
npm update
npm start
npm run make
```
